# Security Policy

## Supported Versions

We actively provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

---

## Reporting a Vulnerability

We take the security of our software very seriously. If you find a security vulnerability, please do not report it in a public issue. Instead, report it privately to the maintainers:

1. **Email us**: Send a detailed report to `apnaafnan27@gmail.com`.
2. **Details**: Include a description of the vulnerability, potential exploit steps, and how it impacts system security.
3. **Response**: We will acknowledge receipt of your report within 48 hours and work with you to release a security fix.

Thank you for helping us keep our code safe and premium!
